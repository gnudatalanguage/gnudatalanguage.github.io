Downloads
=========

Source Code
-----------

- Stable

  - Please visit https://github.com/gnudatalanguage/gdl/releases.
  
- Unstable

  - https://github.com/gnudatalanguage/gdl/archive/master.zip
  
Binaries
--------

- Linux

  - Ubuntu/Debian: :code:`apt install gnudatalanguage`
  - Fedora: :code:`yum install gdl`
  - Gentoo: :code:`emerge gdl`
  - Arch: :code:`pacman -S gnudatalanguage`
    
- macOS

  - Homebrew: :code:`brew install gnudatalanguage`
  - Macports: :code:`port install gnudatalanguage`
  - Fink: :code:`fink install gdl`
    
- Windows

  - Nightly builds can be downloaded at: https://github.com/gnudatalanguage/gdl/actions
