Binaries¶

  • Linux

    • Ubuntu/Debian: apt install gnudatalanguage

    • Fedora: yum install gdl

    • Gentoo: emerge gdl

    • Arch: pacman -S gnudatalanguage

  • macOS

    • Homebrew: brew install gnudatalanguage

    • Macports: port install gnudatalanguage

    • Fink: fink install gdl

  • Windows

    • Nightly builds can be downloaded at: https://github.com/gnudatalanguage/gdl/actions
