Known issues¶

  • In presence of Anaconda (a python installer) ther may be some troubles compiling GDL, see here

  • old GCC. It is still possible to compile GDL 1.0 versions (rc1, rc2, rc3) with old GC 4.8 versions (e.g. gcc version 4.8.4 (Ubuntu 4.8.4-2ubuntu1~14.04.4) but it may give few troubles on few numerics, as shown by make check

  • up to now we never solved the fact that on some computers on long int (Long64, ULong, Ulong64 …) test_fix.pro may fail or not
